Breakfast-Rate

Breakfast Rate is an app that let's users view, share, and comment on their favorite breakfast recipes. This was my first big project focusing on back-end development.

  • Node
  • Mongo
  • Express
  • Restful Routing
  • Forms
  • jQuery
  • User Registration
  • HTML5
  • CSS3
  • Bootstrap
  • Responsive Layout

Some key takeaways:

  • I used the passport.js middleware to create a user registration and login system.
  • I created a middleware to make sure only logged in users can add a recipe or comment, and I created a middleware used for comment and recipe authentication (allows only the owner to edit).
  • I focused on making the recipe forms user-friendly. I used some jQuery fade effects to add a pleasing experience when a user adds ingredients and instructions. I used the jqueryvalidation plugin to notify the user of missing input fields.
  • When a user visits the specific show page of a recipe, there is a sidebar which retrieves three recipes from the database and displays them.
  • I used the masonry library to create a cascading grid system for the index page.

Pizza Builder

Using this app, users select/deselect ingredients and corresponding images appear/disappear on the pizza. After they're done, they can checkout and see the price they owe.

  • Javascript
  • jQuery
  • Responsive Layout
  • HTML5
  • CSS3
  • Vector Graphics

Some key takeaways:

  • In my javascript file, I focused on seperating my concerns more effectively.
  • I created a modal using just plain css and html. I used a nice css transition effect to fade it into view.
  • I had a lot of practice using vector graphics. I used Inkscape (free software similar to Illustrator). I manipulated downloaded svg files and used layers for each pizza topping.
  • I used custom downloaded fonts. Previously I had only used google fonts.
  • I didn't use any framework like Bootstrap to make this site mobile. I used flexbox in my css file.
